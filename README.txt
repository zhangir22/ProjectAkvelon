Документация------------------------------------------------------------------------

Что из себя представляет мой проект

Он создан из системы CRUD-операций
----------------------------------------------------------------------
CRUD - (Create, Read, Update, Delete),--------------------------------
 то есть получение данных, создание, обновление и удаление.-----------
 Entity Framework позволяет легко производить данные операции.--------
----------------------------------------------------------------------
В моем проекте присуствует все эти операции

Но также был создан метод миграции баз данных Под название Code First Migrations

----------------------------------------------------------------------
Code First Migrations ------------------------------------------------
1. Создание исходной базы данных, которая работает с моделью EF-------
2. Создание миграций для отслеживания изменений в модели EF-----------
3. Поддержание актуальности базы данных с учетом этих изменений-------
----------------------------------------------------------------------
